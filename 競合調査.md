## 2. 各サービス詳細

### マネーフォワードME
- **強み**: 2,500+金融機関連携、自動仕訳、家計簿+資産管理の統合、ユーザー基盤が巨大
- **弱み/不満点**:
  - 無料版が連携4件に制限（改悪で不満多数）
  - 連携解除で過去データ消失・復旧不可
  - 口座再連携・同期不良が頻発
  - 二重計上（カード+オンライン購入）の手動除外が必要
  - PayPay等主要QR決済と未連携
  - レシートOCR精度が低い
  - オフライン利用不可
  - 2025年8月に料金値上げ
- **料金**: 無料 / スタンダード月540円 / 資産形成アドバンス月980円

### Moneytree
- **強み**: 無料で50件連携、広告なし、AI自動分類、シンプルUI
- **弱み/不満点**:
  - 投資分析機能が弱い（配当管理・セクター分析なし）
  - 家計簿寄りで株式投資家には物足りない
  - 有料プランの機能差が小さい
- **料金**: 無料 / Grow月360円 / Work月500円

### カビュウ
- **強み**: 株式特化の分析・可視化、ヒートマップ、投資家偏差値、複数口座横断管理
- **弱み/不満点**:
  - 月額1,080円〜と高い（不満の声多数）
  - 無料版は1口座のみで実質有料必須
  - 米国株は有料のみ
  - 投資信託の一部非対応
  - 一般口座は動作保証外
  - みずほ証券等の一部未対応
  - インフルエンサータイアップで信頼性低下の声
- **料金**: 無料 / プライム月1,080円 / ダイヤモンド月2,480円

### ロボフォリオ
- **強み**: 基本無料、適時開示通知、スクリーニング、AI銘柄提案、対応証券会社が多い
- **弱み/不満点**:
  - UIがやや古い
  - 分析の深さはカビュウに劣る
  - モバイルアプリのみ（Web版なし）
- **料金**: 基本無料

### SBI証券 My資産 / 楽天証券 ポートフォリオ
- **強み**: 口座保有者は無料、リアルタイム反映、自社口座の完全連携
- **弱み/不満点**:
  - 自社口座のみ（他社口座は横断管理不可）
  - 分析機能は限定的
  - 複数証券を使う投資家には不十分
- **料金**: 無料（口座保有者）

### Yahoo Finance
- **強み**: グローバル対応、ニュース統合、広範な銘柄カバー、ポートフォリオ分析ツール
- **弱み/不満点**:
  - 日本株のデータが弱い
  - 日本語UIが不十分
  - 有料プラン(Bronze〜Gold)は米国市場向け
- **料金**: 無料 / Bronze / Silver / Gold（米ドル建て）

## 3. FORCE CAPITAL のポジショニング

### 強み（差別化ポイント）
1. **Claude AI 総評** - 競合にない独自機能。ポートフォリオ全体をAIが分析・コメント
2. **日米株横断管理** - J-Quants + yfinance で日本株・米国株を一元管理
3. **シミュレーション機能** - 競合の多くが持っていない
4. **世界指標 + セクター分析 + 配当管理** - 株式特化の分析を無料で提供
5. **シンプルな構成** - Streamlitで軽量、証券口座ログイン不要（セキュリティ面の安心感）

### 弱み（率直な課題）
1. **証券口座自動連携なし** - 手動入力が必要（最大のハードル）
2. **モバイルアプリなし** - Web only（Streamlitの制約）
3. **投資信託・暗号資産 非対応**
4. **家計簿機能なし** - 株式投資に特化（マネフォ/Moneytreeとは棲み分け）
5. **個人開発** - 信頼性・継続性の担保が課題

### 市場での立ち位置
```
家計簿寄り ←――――――――――――――→ 株式投資特化

  Moneytree  マネフォME        ロボフォリオ  カビュウ
                                         ↑
                                   FORCE CAPITAL
                                   (AI総評で差別化)
```

## 6. 機能検討: ベンチマーク比較（カビュウ「投資家偏差値」の代替）

### カビュウの投資家偏差値（参考）
- 全ユーザーの投資データを母集団に、7項目（資産総額/単月最大リターン/取引最大リターン/ペイオフレシオ/勝率/利益率/総合）を偏差値化
- 直近2年間データ、毎月1日更新、プライム限定
- 課題: 母集団が有料ユーザーのみで偏りあり、資産総額の影響が大きすぎるとの声

### FORCE CAPITALでの実装案: 市場ベンチマーク比較
β段階ではユーザー数が少なく偏差値の母集団が成立しないため、**市場指数との比較**で投資成績を可視化する方向が現実的。

| 指標 | 内容 | データソース |
|---|---|---|
| vs TOPIX | 日本株部分のリターンをTOPIXと比較 | J-Quants / yfinance |
| vs S&P 500 | 米国株部分のリターンをS&P500と比較 | yfinance |
| vs 60/40ポートフォリオ | 株60%+債券40%の伝統的配分と比較 | yfinance |
| シャープレシオ | リスク調整後リターン（無リスク金利比） | 計算エンジン(calc.py) |
| 最大ドローダウン | 最高値からの最大下落率 | 計算エンジン(calc.py) |

### 実装優先度
- **Phase 2（β）**: vs TOPIX / vs S&P 500 の単純リターン比較チャート → tab_analysis.py に追加
- **Phase 3以降**: シャープレシオ、最大ドローダウン、ユーザー間比較（偏差値的機能）
